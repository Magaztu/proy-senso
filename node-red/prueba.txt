Para probar el broker idependientemente:
En una terminal:

mosquitto_sub -h localhost -p 1883 -t sensors/data
(Suscripción)

En otra terminal:
mosquitto_pub -h localhost -p 1883 -t sensors/data -m '{"temperature":25,"humidity":50,"timestamp":"2025-06-11T23:59:59Z"}'
(Publicación)

Para probar el sqlite:
sqlite3 backend/data/sensors.db

.tables
SELECT * FROM sensor_data ORDER BY id DESC LIMIT 5;
